🛠️ Hands-On Exercise

Challenge: Prompt Refinement

Start with the naive prompt below. Iteratively improve it using patterns from this chapter until the output quality is excellent.

❌ NAIVE PROMPT
"Write me an email about a project delay"
✅ YOUR GOAL
Produce a professional, empathetic email that a PM could actually send — with specific details and next steps.
⚡ Steps to Follow
1. Add a Role — "You are a senior project manager…"
2. Add Context — project name, 2-week delay, reason, stakeholders
3. Specify Format — tone, length, structure (greeting, body, next steps, closing)
4. Add a Few-shot example of a good delay email
5. Use Chain-of-Thought — "Think step by step about what the recipient needs to know"
