Advanced Techniques

System Prompts
Hidden instructions that define the model's persona, rules, and guardrails. Users never see them.
SYSTEM: You are a helpful assistant for Acme Corp. Never discuss competitors. Always respond in JSON format.
Role Playing
Assign a specific expert role to improve output quality. The model adapts vocabulary, depth, and style.
Act as a cybersecurity auditor with 15 years of experience…
Output Formatting
Specify exactly what structure you want: JSON, Markdown table, bullet points, YAML, XML…
Return a JSON object with keys: title, summary, score (1-10), tags[]
Meta-Prompting
Ask the model to improve its own prompt. Iterate for better results.
Rewrite this prompt to be clearer, more specific, and get better results…
