FEED-FORWARD — non-linear feature transformation
Each token's vector passes independently through two dense layers with a ReLU activation. This adds the model's "knowledge".
Input (d=8)
→
Hidden (4×d)
ReLU: max(0, x)
→
Output (d=8)
Key idea: The FFN acts like a "knowledge store" — attention figures out what to look at; the FFN transforms what it means. In GPT-4 class models, FFN parameters are ~⅔ of total weights.
