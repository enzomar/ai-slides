Running LLMs
Locally

Why run locally?
  • 🔒 Privacy — data never leaves your machine
  • 📶 Offline — no internet connection required
  • 💰 Free — no API costs after setup
  • 🔧 Dev-friendly — custom fine-tunes, no rate limits
Quick start — Ollama
# install (macOS / Linux)
brew install ollama
# pull & run a model
ollama run llama3.2
ollama run mistral
# OpenAI-compatible API
ollama serve # → localhost:11434
