A Typical
AI Workflow

💬 Prompt
User or app
sends request
→
🔤 Tokenize
Text → token IDs
~¾ word each
→
🧠 LLM Inference
Attention + FFN layers
generate probabilities
→
🛠️ Tools / RAG
Search, run code,
call APIs (optional)
→
📄 Output
Text, JSON,
code, image…
→
👤 Human
Validation
Review, verify,
approve or refine
⟳ Iterate — refine the prompt based on output quality until the result is satisfactory
Simple workflow (no tools)
User prompt
→
LLM
→
Response

E.g. ChatGPT, Claude.ai — ideal for writing, summarisation, Q&A.

Agentic workflow (with tools)
Prompt
→
LLM
→
Tool call
→
Result
→
LLM
→
Answer

E.g. RAG pipeline, Copilot, autonomous agents — iterate over external data.
