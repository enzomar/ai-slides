🛠️ Hands-On Exercise

Challenge: AI Pair Programming

Use an AI coding assistant to build a small utility. Compare the AI-generated code against what you would write manually.

📋 TASK

Build a CLI tool that reads a CSV file and outputs a summary: row count, column names, and the top 5 values of a user-specified column.

⚡ Process
1. Describe the task to Copilot Chat / Cursor / Claude
2. Review the generated code — does it handle errors? Edge cases?
3. Ask the AI to add tests, then run them
4. Refactor: ask for improvements — "Make this production-ready"
Reflection: What did the AI get right on the first try? What required human correction? Would you trust this code without review?
